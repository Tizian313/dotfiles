# dotfiles

## About

This repository contains my personal dotfiles and a setup script for configuring my development environment.

These files may overwrite existing configurations. Back up your current dotfiles before using.

## Setup

Clone the repository and navigate into the directory:
```sh
git clone https://github.com/Tizian313/dotfiles.git
cd dotfiles
```

If needed, make the setup script executable:
```sh
chmod +x setup.sh
```

Run setup script:
```sh
./setup.sh
```

## Included Configurations

- **.bashrc** - Bash shell configuration including aliases and function.
- **.inputrc** - Readline configuration for improved command-line editing and history search.
